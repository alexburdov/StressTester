# Проверка backend Rest решений на нагрузку
## Основные принципы
Данный сервис написан на Си по причине получить максимальное быстродействие 

## Методология проверки.

Проверка осуществляется в 2 режимах
1. Получить максимально-возможное количество потоков 
2. Получить значения времени отклика на запрос.

Настройку осуществляется через установка Define 
    https://github.com/alexburdov/StressTester/blob/main/HttpStressClientC/include/common.h
